

# Sales Performance Dashboard 2024

## Project Overview
An interactive Sales Performance Dashboard built in Power BI analysing 500 sales transactions across 5 regions, 6 salespersons and 4 product categories for the year 2024.
This project demonstrates end-to-end data analysis — from raw data to actionable business insights — using real-world sales data.

## Business Questions Answered
- Which region generates the most revenue?
- Which products are top sellers?
- How does revenue trend month by month?
- Which salesperson performs best?
- What is the overall profit margin?

 ## Key Results
- Total Revenue: $434K
- Total Profit: $109K
- Total Orders: 500
- Profit Margin: 25.20%

- ## Key Insights
- North region leads with $108K in revenue
- Furniture and Technology dominate at 44% each
- Yaw Darko is the top salesperson at $91K
- April and October were the strongest months
